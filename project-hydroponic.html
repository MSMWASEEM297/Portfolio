Final Year Project • 2026

Real-Time Automated Hydroponic Cultivation System using IoT and Sensors

An IoT-based hydroponic automation system designed to monitor growing conditions, coordinate pumps and valves, and support web/mobile monitoring.

IoTESP32Raspberry Pi 5MQTTServer.jsDashboard
Main Function

Automated hydroponic monitoring and control

Monitors pH, TDS, water level, temperature and humidity while controlling pumps, valves and nutrient dosing.

01

Project Scope

The project focuses on a low-cost automated hydroponic cultivation system for monitoring and controlling pH, TDS, water level, temperature, humidity and actuator states. The system combines ESP32 sensor/control nodes with a Raspberry Pi 5 local server for real-time data communication, dashboard visualization and automation logic.

02

My Role

My main contribution included Raspberry Pi 5 local server configuration, Mosquitto MQTT broker setup, Server.js backend development, JSON payload processing, dashboard integration, mobile application support, Docker deployment, VNC access, Cloudflare/domain-based remote access and support for hardware integration/testing.

03

Technical Implementation

  • Configured ESP32 nodes to publish sensor and actuator data through MQTT.
  • Used Raspberry Pi 5 as the local server and central controller.
  • Processed JSON telemetry data through a Server.js backend.
  • Displayed separate ESP32 reading cards on the web dashboard.
  • Implemented control concepts for valve-pump coordination, pH-based nutrient dosing, manual mode and emergency stop.
04

Results

The system successfully integrated ESP32 nodes, Raspberry Pi server, MQTT communication, backend logic, dashboard visualization, mobile monitoring and relay-controlled actuators. It improved system visibility by showing each ESP32 node separately and supported automatic hydroponic control operations.
