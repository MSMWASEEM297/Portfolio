# Mohamed Waseem Portfolio Website

This is a simple GitHub Pages portfolio website.

## Files

- `index.html` - main website content
- `style.css` - website design
- `resume.pdf` - replace this with your final resume PDF
- `images/` - replace placeholder images with your real profile and project images

## How to publish

1. Create a GitHub repository named `yourusername.github.io`.
2. Upload all these files into that repository.
3. Go to Settings > Pages.
4. Select branch: `main`, folder: `/root`.
5. Save and wait for GitHub to publish your site.

Your website link will be:

`https://yourusername.github.io`
