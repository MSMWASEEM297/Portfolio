Design Project • 2025

Armyworm Pest Detection and Ultrasonic Repeller System

An intelligent agricultural prototype for camera-based pest detection and non-chemical ultrasonic repelling in cornfields.

Raspberry Pi 5CameraOpenCVTensorFlow LiteGPIOUltrasonic
Main Function

Detect pest and activate ultrasonic repeller

Uses image recognition to detect armyworms and GPIO control to trigger an ultrasonic pest repelling circuit.

01

Project Scope

The project aimed to create a practical and eco-friendly system for early armyworm detection in cornfields. It combines image-based recognition with an ultrasonic repelling mechanism to reduce dependence on manual monitoring and chemical pesticides.

02

My Role

My involvement included supporting system design, prototype development, Raspberry Pi 5 integration, camera-based image recognition workflow, GPIO control, ultrasonic repeller circuit testing and documentation of the detection-to-response process.

03

Technical Implementation

  • Used Raspberry Pi 5 as the main controller for image capture and actuation.
  • Integrated a camera module for image-based pest detection.
  • Used Python, OpenCV and TensorFlow Lite / Teachable Machine concepts for detection.
  • Implemented servo scanning to increase field coverage.
  • Triggered an ultrasonic repeller through GPIO when armyworm detection occurred.
  • Constructed a Colpitts oscillator-based ultrasonic repeller circuit.
04

Results

The prototype demonstrated a complete detection-to-response cycle by identifying armyworm images and activating the ultrasonic repeller. The system provided a sustainable proof-of-concept for low-cost pest control suitable for small- and medium-scale agricultural applications.
